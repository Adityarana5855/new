# Personal Portfolio UI

A clean, responsive personal portfolio website built with HTML, CSS, and vanilla JavaScript. It presents a modern developer profile with animated sections, project highlights, skills, and contact links.

## Preview

This project is a single-page portfolio for **Aditya Pratap Singh**, a Computer Science & Engineering student. The UI uses a dark glassmorphism style, smooth scroll animations, gradient accents, and a mobile-friendly layout.

## Features

- Responsive single-page portfolio layout
- Fixed navigation with active section highlighting
- Hero, About, Projects, and Contact sections
- Smooth scrolling and reveal-on-scroll animations
- Glassmorphism cards and gradient visual styling
- Skills section for technologies and strengths
- Project cards for showcasing selected work
- Social/profile links for GitHub, LeetCode, and HackerRank
- No build tools or framework required

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts: Outfit
- Phosphor Icons

## Getting Started

Because this is a static website, no installation is required.

1. Clone the repository:

```bash
git clone https://github.com/your-username/portfolio-ui.git
```

2. Open the project folder:

```bash
cd portfolio-ui
```

3. Open `index.html` in your browser.

You can also use the VS Code Live Server extension for a smoother local development experience.

## Customization

Update the following sections in `index.html` to personalize the portfolio:

- Name and headline in the hero section
- About Me content
- Skills list
- Project titles, descriptions, and GitHub links
- Email and social profile links

To change the visual theme, edit the CSS variables in `style.css`:

```css
:root {
    --bg-dark: #09090b;
    --accent-1: #06b6d4;
    --accent-2: #a855f7;
    --accent-3: #ec4899;
}
```

## Sections

- **Home:** Introduction and call-to-action buttons
- **About:** Short background, education, and core skills
- **Projects:** Selected project cards with descriptions
- **Contact:** Email and coding profile links
